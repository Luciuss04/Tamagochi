Proyecto de entornos  de desarollo :

Este proyecto consiste en un Tamagotchi virtual desarrollado en Java.
El Tamagotchi puede realizar las siguientes acciones:

Dormir: Recupera energía pero reduce saciedad y diversión.

Comer: Aumenta la saciedad pero reduce la diversión.

Bañar: Aumenta la diversión y reduce la energía.

Jugar: Aumenta la diversión pero reduce saciedad y energía.

Mostrar Estado: Muestra el nivel actual de saciedad, energía y diversión, junto con un sprite ASCII que refleja su estado emocional.

El juego termina cuando alguna de las estadísticas llega a 0 (Game Over).
